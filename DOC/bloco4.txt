### **Perguntas do Bloco 4**

1.  **Fontes de pesquisa**
    Onde foram pesquisadas as soluções?
      * ( ) Sites oficiais de governo/Painel de Preços
      * ( ) Contratações similares
      * ( ) Internet especializada
      * ( ) Consulta a fornecedores
      * ( ) Outros - Especifique
2.  **Justificativa técnica**
    Forneça análise técnica COMPLETA:
      * Por que esta solução atende à necessidade?
      * Quais alternativas foram avaliadas?
      * Aspectos técnicos diferenciais
      * Conformidade com normas
3.  **Justificativa econômica**
    Forneça análise econômica DETALHADA:
      * Comparação de custos (próprio vs terceirizado)
      * Custo-benefício da solução
      * Economia gerada ou justificativa de valor
      * Previsibilidade orçamentária
4.  **Restrições de mercado**
    Há limitações de fornecedores?
      * ( ) Sim - Explique quais e por quê
      * ( ) Não há restrições conhecidas
5.  **Tratamento ME/EPP**
    Será dispensado tratamento diferenciado?
      * ( ) Sim - Justifique conforme LC 123/2006
      * ( ) Não - Aplicará tratamento normal
