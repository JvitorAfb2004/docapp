### **Perguntas do Bloco 6**

[cite\_start]Aqui estão as perguntas que a IA deve usar para coletar as informações necessárias para a seção de Resultados e Gestão, conforme o documento `Blocos de Perguntas DFD-ETP.pdf`[cite: 22, 171]:

1.  **Benefícios pretendidos**
    Quais resultados espera-se obter?
      * ( ) Manutenção do funcionamento administrativo
      * ( ) Redução de custos operacionais
      * ( ) Aproveitamento de recursos humanos
      * ( ) Redução de riscos do trabalho
      * ( ) Ganho de eficiência operacional
      * ( ) Melhoria na qualidade de produtos/serviços
      * ( ) Realização de política pública específica
      * ( ) Outro - Especifique
2.  **Nota explicativa dos resultados**
    [cite\_start]Forneça uma descrição DETALHADA dos resultados esperados com a contratação, explicando como os benefícios selecionados na pergunta anterior contribuirão para os objetivos institucionais[cite: 22, 183].
3.  **Providências pendentes**
    Há ações necessárias antes da contratação?
      * ( ) Sim - Liste todas
      * ( ) Não há pendências
4.  **Gestão e fiscalização**
    Confirme requisitos específicos:
      * Competências necessárias para gestor
      * Competências necessárias para fiscal
      * Conhecimentos técnicos específicos
      * Frequência de acompanhamento
5.  **Contratações relacionadas**
    Há interdependências?
      * ( ) Sim - Especifique quais contratos/processos
      * ( ) Não há correlações

### **Variáveis e Regras de Preenchimento**

  * **Benefícios pretendidos:**

      * [cite\_start]**Variáveis no ETP:** `{beneficios_pretendidos_manutencao}`, `{beneficios_pretendidos_reducao_custos}`, `{beneficios_pretendidos_aproveitamento}`, `{beneficios_pretendidos_reducao_riscos}`, `{beneficios_pretendidos_ganho_eficiente}`, `{beneficios_pretendidos_qualidade}`, `{beneficios_pretendidos_politica_publica}` e `{beneficios_pretendidos_outro}`[cite: 398].
      * **Regra:** Marcar com um "X" uma ou mais opções. [cite\_start]Se a opção "Outro" for selecionada, preencher a variável `{beneficios_pretendidos_beneficio}` com o benefício específico[cite: 398].

  * **Nota explicativa dos resultados:**

      * [cite\_start]**Variável no ETP:** `{beneficios_pretendidos_explicacao}`[cite: 398].
      * **Regra:** Preencher com um texto detalhado, explicando como os benefícios selecionados contribuirão para os objetivos da instituição.

  * **Providências pendentes:**

      * [cite\_start]**Variáveis no ETP:** `{providencias_pendentes_sim}` e `{providencias_pendentes_nao}`[cite: 399].
      * **Regra:** Marcar com um "X" uma das opções. [cite\_start]Se "Sim", listar as providências na variável `{providencias_pendentes_especificar}`[cite: 399].

  * **Gestão e fiscalização:**

      * [cite\_start]**Variáveis no ETP:** `{requisitos_gestao_contratual}` e `{requisitos_fiscalizacao}`[cite: 399].
      * [cite\_start]**Regra:** Preencher com um texto detalhado que descreva as competências e conhecimentos necessários para a gestão e fiscalização do contrato[cite: 22, 190, 191, 192].

  * **Contratações relacionadas:**

      * [cite\_start]**Variáveis no ETP:** `{contratacoes_correlatas_sim}` e `{contratacoes_correlatas_nao}`[cite: 400].
      * **Regra:** Marcar com um "X" uma das opções. [cite\_start]Se "Sim", especificar os contratos/processos na variável `{contratacoes_correlatas_justificativa}`[cite: 400].
